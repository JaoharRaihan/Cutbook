## 🎨 How to Take Screenshots

### Option 1: From Simulator/Emulator (Recommended)

**Android**:

```bash
# Start emulator
cd ~/Desktop/NAW/CutBook
npx react-native run-android

# Take screenshot from emulator toolbar
# Or use: adb shell screencap -p /sdcard/screenshot.png
```

**iOS**:

```bash
# Start simulator
cd ~/Desktop/NAW/CutBook
npx react-native run-ios

# In simulator: Device → Screenshot (or Cmd+S)
```

### Option 2: From Physical Device

**Android**:

- Power + Volume Down (most devices)
- Screenshots saved to Gallery

**iOS**:

- Side Button + Volume Up (iPhone with Face ID)
- Screenshots saved to Photos

### Option 3: Design Tool (Professional)

Use Figma or similar tools with device frames:

1. Take plain screenshots
2. Add to device mockup
3. Add text/annotations
4. Export at required resolution
